# India EV Market Analysis Dashboard - Tableau Project

### **Comprehensive Visualization of India’s Electric Vehicle Ecosystem (2018-2024)**

---

## **Project Overview**

This Tableau project provides a complete analytical view of India’s Electric Vehicle (EV) market using a **sample dataset generated in Python**.
The dashboard visualizes major EV adoption trends, state-level distribution, manufacturer dominance, model-level insights, and CAFV eligibility.

The goal of this project is to demonstrate strong data visualization, storytelling, and analytical skills using Tableau-while analyzing the rapidly growing EV ecosystem in India.

---

## **Dataset Description**

The dataset used in this project is a **synthetic EV dataset created using Python**, designed to reflect realistic EV adoption trends across India between 2018 and 2024.

### **Key Features in the Dataset**

* **Model Year**
* **Make (Manufacturer)**
* **Model**
* **Electric Vehicle Type** (BEV / PHEV)
* **Electric Range (km)**
* **Base MSRP (₹)**
* **State & City**
* **Postal Code**
* **CAFV Eligibility**
* **Latitude & Longitude**
* **Electric Utility Provider**

### **Cleaning & Preparation Steps**

* Standardized state names
* Corrected missing & inconsistent values
* Ensured valid numerical ranges
* Added region mapping
* Prepared lat/long for geographic charts

### 1. Key Performance Indicators (KPIs)

---

## **Key Insights**

### Market Insights

* BEVs dominate India’s EV landscape (>80%).
* Strong upward trend in EV adoption after 2021.

### Geographic Insights

* Maharashtra, Karnataka, Tamil Nadu, and Delhi lead EV adoption.

### Manufacturer Insights

* Ather, Tata Motors, and TVS dominate the market.

### Policy Insights

* Majority of EVs qualify for CAFV incentives, showing policy impact.
